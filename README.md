## 🌟 项目简介 (Project Overview)

**Shensist Matrix: 偃甲逻辑审计智能体** 是一款专为中小学 AI 教育设计的交互式逻辑学习工具。该项目基于 [Shensist-Agent](https://shensist.top/) 矩阵架构，融合了中式“偃甲”美学（Guofeng Cyberpunk）与因果推断逻辑，通过可视化的“因果碎片”编排，引导学生理解顺序逻辑、条件分支及因果悖论。

The **Shensist Matrix: Logic Causality Agent** is an interactive educational tool designed for primary and middle school students. It combines traditional Chinese "Yan-jia" aesthetics with causal inference logic, allowing students to explore sequential logic and paradoxes by arranging "Causal Shards."

## 🚀 核心特性 (Key Features)

- **L-A-T-T 架构**: 极致轻量化，无后端依赖，纯 HTML5/Vanilla JS/CSS3 实现。
- **深度逻辑审计 (Deep Logic Auditor)**: v7.0 引入的审计引擎，可实时评估因果链的得分、优劣并提供修正建议。
- **演武秘籍 (Academy Tutorial)**: 内置交互式教学手册，适合 3-9 年级学生快速上手。
- **因果对比 (Comparative Wisdom)**: 支流保存与对比系统，支持不同决策路径的差异化分析。
- **国风美学 (Guofeng Aesthetics)**: 锦缎宣纸纹理、灵镜徽标与五声音阶反馈，营造沉浸式学习体验。

## 🛠️ 快速开始 (Quick Start)

1. **下载项目**: 直接下载或克隆本仓库。
2. **运行**: 在浏览器中直接打开 `index.html` 即可，无需任何服务器环境。
3. **操作流程**:
    - **拖拽**: 将上方的因果碎片放入下方的矩阵。
    - **执行**: 点击“执行因果收束”观察偃甲机器人的反应。
    - **审计**: 点击侧边栏快照的“对比”按钮，获取深度逻辑分析。

## 📚 教育目标 (Educational Goals)

- **顺序逻辑**: 理解因果链的前后依赖关系。
- **悖论识别**: 学习如何识别及避免逻辑死循环或状态冲突。
- **多路径优化**: 训练对比不同解决方案并寻找最优解的能力。
